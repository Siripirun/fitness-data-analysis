# Fitness Data Analysis: Activity, Calories & Performance

## Overview
This project analyses Fitbit activity data to explore the relationship between physical activity, calorie expenditure, and performance metrics.

## Objective
To understand how activity levels influence calorie burn and overall fitness behaviour using real-world data.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- Kaggle Notebook
- GitHub

## Analysis Performed
- Steps vs calories analysis
- Activity intensity distribution
- Sedentary behaviour vs calories
- Active minutes vs performance

## Key Insights
- Higher step counts are associated with increased calorie expenditure  
- Activity intensity varies significantly across individuals  
- Both movement and intensity contribute to overall performance
